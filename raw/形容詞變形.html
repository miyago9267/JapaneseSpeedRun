形容詞變化型完全對照表

い形容詞 vs な形容詞・兩種視角切換
否定形 「不～」
類型規則例子
い形 い → くない 高い → 高くない
寒い → 寒くない　／　難しい（むずかしい）→ 難しくない
な形 字根 ＋ じゃない 暇（ひま）→ 暇じゃない
静か（しずか）→ 静かじゃない　／　便利（べんり）→ 便利じゃない
過去形 「曾經是～」
類型規則例子
い形 い → かった 高い → 高かった
寒い → 寒かった　／　楽しい（たのしい）→ 楽しかった
な形 字根 ＋ だった 暇 → 暇だった
静か → 静かだった　／　好き（すき）→ 好きだった
過去否定形 「以前不是～」
類型規則例子
い形 い → くなかった 高い → 高くなかった
忙しい（いそがしい）→ 忙しくなかった
な形 字根 ＋ じゃなかった 暇 → 暇じゃなかった
有名（ゆうめい）→ 有名じゃなかった
て形　連接 「～而且～」／串接多個描述
類型規則例子
い形 い → くて 安い（やすい）→ 安くてうまい
小さい → 小さくて　／　寒い → 寒くて
な形 字根 ＋ で 静か → 静かできれいな部屋（へや）
便利 → 便利で　／　親切（しんせつ）→ 親切で
副詞化 修飾動詞・程度
類型規則例子
い形 い → く 早い（はやい）→ 早く起きる
強く（つよく）押す　／　上手く（うまく）やる
な形 字根 ＋ に 静か → 静かにしてください
丁寧（ていねい）に書く　／　親切に教える
接名詞 直接修飾名詞
類型規則例子
い形 原形 ＋ 名詞 高い山（やま）　／　寒い日（ひ）
うるさい人　／　眠い（ねむい）授業（じゅぎょう）
な形 字根 ＋ な ＋ 名詞 静かな場所（ばしょ）　／　有名な人
便利なアプリ　／　好きな曲（きょく）
たら形　假定 「如果～的話」（具體情境假設）
類型規則例子
い形 い → かったら 高い → 高かったら買わない
安かったら買う　／　暑かったらエアコンつけて
な形 字根 ＋ だったら 暇 → 暇だったら来て
便利だったら使う　／　嫌いだったら残していい

⚠ たら 帶「到那個時間點／如果真的發生」的語感，最口語通用。四種條件的差別見下面比較表。

ば形　條件 「如果～就～」（一般條件／假設）
類型規則例子
い形 い → ければ 高い → 高ければ買わない
安ければ買う　／　寒ければ窓（まど）を閉めて
な形 字根 ＋ なら（ば） 暇 → 暇なら来て
便利なら使う　／　静かなら集中（しゅうちゅう）できる

⚠ いい（好い）は例外：よければ。な形的 ば 一般用「なら」，正式可説「字根 ＋ であれば」（暇であれば）。

條件形比較　たら／ば／なら／と 四種「如果」的語感差別
條件形い形（高い）な形（暇）
たら 高かったら 暇だったら
ば 高ければ 暇なら（ば）
なら 高いなら 暇なら
と 高いと 暇だと

⚠ 語感：たら＝具體情境，最口語通用；ば＝一般條件／假設；なら＝「如果是～的話」帶前提或主題；と＝必然結果「一～就～」。な形的 ば 形其實就用「なら」。

陷阱：看起來像い形、其實是な形 字尾碰巧是い音，但行為像な形容詞
詞錯誤變法正確變法
きれい きれいくない ✗ きれいじゃない ✓　／　きれいな部屋 ✓
嫌い（きらい） 嫌いくない ✗ 嫌いじゃない ✓　／　嫌いな食べ物 ✓
真面目（まじめ） 真面目くない ✗ 真面目じゃない ✓　／　真面目な人 ✓
素直（すなお） 素直くない ✗ 素直じゃない ✓　／　素直な子 ✓
有名（ゆうめい） 有名くない ✗ 有名じゃない ✓　／　有名な人 ✓
丁寧（ていねい） 丁寧くない ✗ 丁寧じゃない ✓　／　丁寧に ✓

⚠ 判斷方式：試著在後面加「な＋名詞」。可以的就是な形容詞。きれいな部屋 → OK → な形。

い形容詞
い形容詞・全變化型 字尾是い・直接改字尾
變化型規則例：高い　／　寒い　／　楽しい（たのしい）
原形
現在肯定		 字典形（不變） 高い　／　寒い　／　楽しい
否定 い → くない 高くない　／　寒くない　／　楽しくない
過去 い → かった 高かった　／　寒かった　／　楽しかった
過去否定 い → くなかった 高くなかった　／　寒くなかった　／　楽しくなかった
て形 い → くて 高くて　／　寒くて　／　楽しくて
副詞化 い → く 高く　／　寒く　／　楽しく
接名詞 原形＋名詞 高い山　／　寒い夜（よる）　／　楽しい時間（じかん）
たら形 い → かったら 高かったら　／　寒かったら　／　楽しかったら
ば形 い → ければ 高ければ　／　寒ければ　／　楽しければ

⚠ いい（好い）は例外：否定 → よくない　過去 → よかった　副詞 → よく　條件 → よければ。「いくない」は存在しない。

な形容詞
な形容詞・全變化型 字根不動・靠だ屈折
變化型規則例：暇（ひま）　／　静か（しずか）　／　便利（べんり）
原形
現在肯定		 字根 ＋ だ 暇だ　／　静かだ　／　便利だ
否定 字根 ＋ じゃない 暇じゃない　／　静かじゃない　／　便利じゃない
過去 字根 ＋ だった 暇だった　／　静かだった　／　便利だった
過去否定 字根 ＋ じゃなかった 暇じゃなかった　／　静かじゃなかった　／　便利じゃなかった
て形 字根 ＋ で 暇で　／　静かで　／　便利で
副詞化 字根 ＋ に 暇に（較少用）　／　静かに　／　便利に
接名詞 字根 ＋ な ＋ 名詞 暇な時間　／　静かな場所　／　便利なアプリ
たら形 字根 ＋ だったら 暇だったら　／　静かだったら　／　便利だったら
ば形 字根 ＋ なら（ば） 暇なら　／　静かなら　／　便利なら
