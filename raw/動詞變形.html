動詞變化型完全對照表

五段 vs 一段 vs 不規則・兩種視角切換
先辨認動詞屬於哪一類 下面每個變化型都分五段／一段／不規則，先會分類才套得了規則
類型怎麼認例
一段
（る動詞）		 字尾る，且る前是い段／え段音 見る　／　食べる　／　起きる
寝る（ね）　／　教える（おし）
五段
（う動詞）		 字尾是う段：く・ぐ・す・つ・ぬ・ぶ・む・る・う 書く　／　泳ぐ　／　話す　／　待つ
飲む　／　買う　／　帰る（る前是え段，仍是五段）
不規則 只有兩個，整組背 する　／　くる（来る）

⚠ 陷阱：字尾る＋前面是い／え段音「看起來像一段」，但這些其實是五段 → 帰る・入る（はいる）・走る（はしる）・要る（いる）・知る（しる）・切る（きる）。同音不同類：切る（きる・五段）vs 着る（きる・一段）。分不出來就查字典。

ない形　否定 「不做～」
類型規則例子
五段 字尾 → あ段 ＋ ない 書く → 書かない
飲む → 飲まない　／　待つ → 待たない　／　買う → 買わない（あ段用わ）
一段 去る ＋ ない 食べる → 食べない
起きる → 起きない
不規則 單獨記 する → しない　／　くる → こない
ます形　禮貌肯定 「做～（禮貌）」／ ません＝禮貌否定
類型規則例子
五段 字尾 → い段 ＋ ます 書く → 書きます
飲む → 飲みます　／　待つ → 待ちます
一段 去る ＋ ます 食べる → 食べます
起きる → 起きます
不規則 單獨記 する → します　／　くる → きます
て形　連接／請求／進行 接續其他句型的基礎形
類型規則例子
五段
（く）		 く → いて 書く → 書いて
五段
（ぐ）		 ぐ → いで 泳ぐ → 泳いで
五段
（す）		 す → して 話す → 話して
五段
（む・ぬ・ぶ）		 → んで 飲む → 飲んで　／　遊ぶ → 遊んで
五段
（る・う・つ）		 → って 帰る → 帰って　／　買う → 買って　／　待つ → 待って
一段 去る ＋ て 食べる → 食べて　／　起きる → 起きて
不規則 單獨記 する → して　／　くる → きて

⚠ 例外：行く → 行って（不是行いて）

た形　過去式 「做了～」／規則與て形完全相同，て換成た
類型規則例子
五段（く） く → いた 書く → 書いた
五段（ぐ） ぐ → いだ 泳ぐ → 泳いだ
五段（す） す → した 話す → 話した
五段（む・ぬ・ぶ） → んだ 飲む → 飲んだ　／　遊ぶ → 遊んだ
五段（る・う・つ） → った 帰る → 帰った　／　買う → 買った　／　待つ → 待った
一段 去る ＋ た 食べる → 食べた　／　起きる → 起きた
不規則 單獨記 する → した　／　くる → きた

⚠ 例外：行く → 行った（不是行いた）

ば形　條件 「如果做～的話」
類型規則例子
五段 字尾 → え段 ＋ ば 書く → 書けば
飲む → 飲めば　／　待つ → 待てば
一段 去る ＋ れば 食べる → 食べれば　／　起きる → 起きれば
不規則 單獨記 する → すれば　／　くる → くれば
可能形　能力 「做得到～／能做～」
類型規則例子
五段 字尾 → え段 ＋ る 書く → 書ける
飲む → 飲める　／　待つ → 待てる
一段 去る ＋ られる 食べる → 食べられる　／　起きる → 起きられる
不規則 單獨記 する → できる　／　くる → こられる

⚠ するの可能形是できる，不是しられる

意向形　意志／邀請 「（我）要做～吧／一起做～吧」
類型規則例子
五段 字尾 → お段 ＋ う 書く → 書こう
飲む → 飲もう　／　待つ → 待とう
一段 去る ＋ よう 食べる → 食べよう　／　起きる → 起きよう
不規則 單獨記 する → しよう　／　くる → こよう
受身形　被動 「被～／受到～」
類型規則例子
五段 字尾 → あ段 ＋ れる 書く → 書かれる
飲む → 飲まれる　／　待つ → 待たれる
一段 去る ＋ られる 食べる → 食べられる　／　起きる → 起きられる
不規則 單獨記 する → される　／　くる → こられる

⚠ 一段的可能形和受身形都是られる，靠語境判斷

使役形　使役 「讓～做／叫～做」
類型規則例子
五段 字尾 → あ段 ＋ せる 書く → 書かせる
飲む → 飲ませる　／　待つ → 待たせる
一段 去る ＋ させる 食べる → 食べさせる　／　起きる → 起きさせる
不規則 單獨記 する → させる　／　くる → こさせる
五段動詞
先懂原理：字尾在「自己那一行」的あ→お間移動 以 書く（か行：か・き・く・け・こ）為例
五十音段書く 變成這個段位接哪些變化型
あ段 書か ない形・受身・使役
書かない　／　書かれる　／　書かせる
い段 書き ます形
書きます
う段 書く 辞書形（原形・字典上的樣子，不變）
え段 書け ば形・可能形
書けば　／　書ける
お段 書こ 意向形
書こう

⚠ 換動詞就換成「它自己那一行」：待つ → た行（た・ち・つ・て・と）、飲む → ま行（ま・み・む・め・も）。う 結尾的動詞 あ段 用「わ」：買う → 買わない（不是買あない）。て形／た形另外走音便，見下表。

述語的肯定／否定 × 時制
五段（例：書く） 同一個動詞・四種時制文體 × 肯否
時制・文體肯定否定
現在・普通 書く 書かない
現在・丁寧 書きます 書きません
過去・普通 書いた 書かなかった
過去・丁寧 書きました 書きませんでした
一段（例：食べる） 去る再接，沒有音便
時制・文體肯定否定
現在・普通 食べる 食べない
現在・丁寧 食べます 食べません
過去・普通 食べた 食べなかった
過去・丁寧 食べました 食べませんでした
不規則（する） 整組單獨記
時制・文體肯定否定
現在・普通 する しない
現在・丁寧 します しません
過去・普通 した しなかった
過去・丁寧 しました しませんでした
不規則（くる・来る） 讀音 こ／き 跟著切換
時制・文體肯定否定
現在・普通 くる こない
現在・丁寧 きます きません
過去・普通 きた こなかった
過去・丁寧 きました きませんでした

普通形＝常體（朋友・家人・日記）；丁寧形＝です・ます體（對外・正式）。否定的「過去」是 ない → なかった、ません → ませんでした。

其他變化型的否定
其他變化型怎麼變否定 以 書く 為例
變化型肯定否定
て形 書いて 書かなくて　／　書かないで
ば形 書けば 書かなければ
可能形 書ける 書けない
受身形 書かれる 書かれない
使役形 書かせる 書かせない

⚠ て形否定有兩種：〜なくて＝原因／狀態（書かなくて困る）；〜ないで＝請求／不做某事（書かないでください）。可能・受身・使役本身是一段型，否定一律「去る ＋ ない」。
