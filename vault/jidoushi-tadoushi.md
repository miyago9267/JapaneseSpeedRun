# 自他動詞

日文動詞分兩種性格：**他動詞**（我對東西做某事・有對象）、**自動詞**（東西自己發生了某事・無主動施者）。助詞選錯・ている 意思就完全跑掉。延伸：[動詞變化型](verb-conjugation.md)。

## 核心差異

| | 他動詞（たどうし） | 自動詞（じどうし） |
|---|---|---|
| 是什麼 | 施者「對某物」做的動作 | 某事「自行」發生的狀態 |
| 搭配助詞 | `を`（直接賓語） | `が`（主語自行發生） |
| 典型例 | 窓**を**開ける（我開窗） | 窓**が**開く（窗開了） |

> [!WARNING]
> 助詞就是關鍵：を → 他動詞；が → 自動詞。寫を但用了自動詞（窓を開く）、或寫が但用了他動詞（窓が開ける）都算錯。

## 形狀辨認：常見配對規律

大部分自他動詞是成對的，字形有規律可循。

| 規律 | 他動詞（を） | 自動詞（が） |
|---|---|---|
| `〜ける ／ 〜く` | 開**ける**・続**ける** | 開**く**・続**く** |
| `〜す ／ 〜る` | 出**す**・起こ**す**・落と**す**・消**す** | 出**る**・起き**る**・落ち**る**・消え**る** |
| `〜える ／ 〜わる` | 変**える**・終**える**・伝**える**・集め**る** | 変**わる**・終**わる**・伝**わる**・集**まる** |
| `〜れる ／ 〜る` | 入れ**る**・乗せ**る** | 入**る**・乗**る** |
| 不規則 | 見**せる**・着**る**（自他判斷靠語境） | 見**える**・聞こえ**る** |

> [!WARNING]
> 消す（他）vs 消える（自）看起來不像一對，但就是同義對：「我消掉它」vs「它消失了」。す結尾 = 幾乎都是他動詞。

## ている 的意思：自他決定一切

接 ている 時，自・他的選擇直接改變語義：

| 句型 | 例句 | 意思 |
|---|---|---|
| **他動詞** + ている | ドア**を**開け**ている** | 「正在開門」（動作進行中） |
| **自動詞** + ている | ドア**が**開い**ている** | 「門是開著的」（結果狀態） |
| 他動詞 + ている | 電気**を**消し**ている** | 「正在關燈」 |
| 自動詞 + ている | 電気**が**消え**ている** | 「燈是熄的」（已熄滅狀態） |

> [!WARNING]
> 「窓が開いている」≠「有人現在正在開窗」，是「窗子處於打開的狀態」。要說「有人在開窗」必須用 窓を開けている。N2 作文最常見的語義混淆就在這。

## 沒有配對的動詞（只有其中一個）

少數動詞只存在自動詞形或只存在他動詞形：

- **只有他動詞**：食べる・飲む・読む・書く（本來就是對外做的事）
- **只有自動詞**：ある・いる・来る（存在狀態）
- 要強制「讓它發生」→ 用使役形 食べさせる・飲ませる

## 快速判斷法

1. 有施者對某物做事 → 他動詞 ＋ を
2. 東西自行發生／狀態改變 → 自動詞 ＋ が
3. 看字尾：**す** 結尾幾乎都他動詞；**〜わる・〜まる・〜ける（→く)・〜る** 結尾多自動詞
4. 不確定 → 查字典找「他」「自」標記
