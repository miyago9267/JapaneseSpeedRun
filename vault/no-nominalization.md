# の 接續與名詞化

の 有多個功能。這課專注「**名詞化の**」——把動詞片語包成名詞，再接各種助詞——以及容易混淆的のに・ので・のだ。延伸：[動詞變化型](verb-conjugation.md)。

## の 的名詞化功能

`動詞普通形 ＋ の` → 整個動作包成「一件事」，可以接任何名詞位置的助詞。

| 接的助詞 | 功能 | 例句 | 中文 |
|---|---|---|---|
| **のが** | 主語（行為本身是主角） | 泳**ぐのが**好きだ | 我喜歡游泳 |
| **のは** | 主題（對比・強調話題） | 早起き**するのは**難しい | 早起這件事很難 |
| **のを** | 直接賓語（看到・忘記某事） | 彼が泣く**のを**見た | 我看到他在哭 |
| **のに**（目的） | 「為了做～需要」 | 運転**するのに**免許が要る | 開車需要駕照 |
| **のと**（比較） | 並列兩個動作比較 | 歩く**のと**走る**のと**どちらが速い？ | 走路和跑步哪個快？ |

> [!WARNING]
> のを 的動作一定是「主語親眼看到／感知到」的事：見る・聞く・感じる・知る 等感知動詞做述語。若是抽象事實，用 ことを。

## のは vs のが — 選哪個？

兩個都能說「游泳這件事」，差在結構重心：

| | 句型 | 強調的是 |
|---|---|---|
| **のが** | 泳ぐ**のが**好きだ | 「游泳」是主語，好きだ 是述語 → 說明這行為的屬性 |
| **のは** | 泳ぐ**のは**好きだ**が**、走るのは嫌いだ | は 引入對比 → 強調「游泳這件事」vs 其他事 |

簡單說：沒有對比對象時用のが；有對比・轉折時用のは。

## のに（逆接）— 和目的用法完全不同

のに 接在**句子**後面表示「明明⋯卻⋯（不符合預期）」，這個の是接詞，不是名詞化の。

| | 例句 | 中文 |
|---|---|---|
| 逆接 | 頑張った**のに**、合格できなかった | 明明很努力，卻沒考上 |
| 逆接 | 安い**のに**美味しい | 明明便宜，卻很好吃（な形・い形直接接のに） |
| 目的（名詞化） | 料理する**のに**時間がかかる | 做料理這件事需要時間 |

> [!WARNING]
> 判斷是哪種：前後句如果是「預期落空」就是逆接のに；如果後面是「需要・使用・花費」就是目的のに。

## ので（原因）— 說明性・客觀

のでも把前面名詞化後表示原因，語感比から更客觀・有禮貌。

| | 例句 | 中文 |
|---|---|---|
| ので | 雨が降った**ので**、傘を持った | 因為下雨，所以帶傘 |
| ので | 風邪を引いた**ので**、休みます | 因為感冒了，所以請假（客氣） |

> [!WARNING]
> ので vs から：のでは客觀理由（自然因果）；から 是主觀理由或口語命令理由（やるから見てて！）。日檢作文用のでは會顯得更自然。

## のだ ／ んだ ／ のです — 解釋・說明

`普通形 ＋ のだ` = 「事情是這樣的」，給已知情況提供解釋或強調。

| 文體 | 形式 | 場合 |
|---|---|---|
| 書面・正式 | 〜**のです** ／ 〜**のだ** | 解釋為什麼 |
| 口語 | 〜**んです** ／ 〜**んだ** | 日常對話 |

例：

- 具合が悪い**んです**。— 「我其實不太舒服」（解釋自己狀況）
- あ、それが答えな**んだ**！— 「原來那就是答案啊！」（理解・恍然大悟）
- なぜ遅れた**んですか**？— 「你到底是為什麼遲到的？」（追問原因）

> [!WARNING]
> のです 不只是禮貌語氣，它帶有「我在解釋・說明背景」的含義。純粹的陳述不加のだ；加了就是在解釋或要求對方理解。

## の vs こと — 名詞化兩兄弟

兩個都能名詞化動詞，但使用場景不同：

| | の | こと |
|---|---|---|
| 感知動詞（見る・聞く） | ✓ 彼が笑う**のを**見た | ✗ 彼が笑う**ことを**見た |
| 好惡・上手 | ✓ 歌う**のが**好き | ✓ 歌う**ことが**好き（皆可） |
| 抽象事實（知る・わかる） | ✓ 彼が来る**のを**知っている | ✓ 彼が来る**ことを**知っている（皆可） |
| ことができる（可能） | ✗ 泳ぐ**のができる** | ✓ 泳ぐ**ことができる** |
| ことがある（經驗・偶爾） | ✗ 食べる**のがある** | ✓ 食べた**ことがある** |
| ことにする（決定） | ✗ 辞める**のにする** | ✓ 辞める**ことにする** |

**判斷捷徑**：

- 後面是 できる・ある・にする・になる → 幾乎都用 **こと**
- 後面是感知動詞（見る・聞く・感じる） → 幾乎都用 **の**
- 好き・嫌い・上手・下手 → 兩個都行，の 更口語
