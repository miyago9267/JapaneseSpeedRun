# 細節助詞：も・しか・だけ・ばかり

這四個助詞都能表示「範圍・限定・強調」，但語感和用法差很多。延伸：[句子解析](sentence-parsing.md)。

## 一眼比較

| 助詞 | 核心語感 | 搭配動詞 | 語氣 |
|---|---|---|---|
| **も** | 「也・甚至・竟然」 | 肯定・否定皆可 | 追加・強調 |
| **しか** | 「只有（不夠）」 | **必須否定** | 不足・遺憾 |
| **だけ** | 「只有（中性）」 | 肯定・否定皆可 | 中性限定 |
| **ばかり** | 「淨是・剛剛・大約」 | 依用法而異 | 過度・剛完成・概數 |

> [!WARNING]
> **しか 必須接否定動詞**：しかある・しか食べた → 錯誤。一看到 しか 就預期後面有 ない／できない／いない。

## も — 三種用法

### 1. 追加「也」
```
私も行く。（我也去。）
彼女も来た。（她也來了。）
```

### 2. 強調意外的量「竟然・居然有這麼多（少）」
```
3時間も待った。（居然等了三個小時。）← 多得出乎意料
100円しか持っていない。← 注意這是しか，不是も
```

### 3. 疑問詞 ＋ も ＋ 否定 → 「一個都沒有」
```
誰もいない。（沒有任何人。）
何も食べなかった。（什麼都沒吃。）
どこにも行かなかった。（哪裡都沒去。）
```

## しか — 「只有」＋必定否定，帶遺憾語氣

```
財布に500円しかない。（錢包裡只有500圓。）← 言下之意：不夠
このバスは1日に2本しか来ない。（這班巴士一天只有兩班。）← 太少了
```

> [!WARNING]
> しか 的否定是「不可或缺」，不是可選的。しかない = 「除此之外都不存在」，不是「沒有限制」。

## だけ — 「只有」中性，不帶情緒

```
コーヒーだけ飲んだ。（只喝了咖啡。）← 中性陳述
見るだけで買わなかった。（只是看看，沒有買。）
これだけあれば十分だ。（有這些就夠了。）← 甚至帶有足夠感
```

### しか vs だけ 對比

| | 句子 | 語感 |
|---|---|---|
| しか | りんごしかない | 只有蘋果（不夠・可惜） |
| だけ | りんごだけある | 只有蘋果（中性事實） |

## ばかり — 三種用法

### 1. 淨是・盡是（過度・抱怨語感）
```
彼はゲームばかりしている。（他整天只知道打電動。）
甘いものばかり食べると体に悪い。（一直吃甜食對身體不好。）
```

### 2. たばかり — 「剛剛做了」
```
日本に着いたばかりだ。（我剛抵達日本。）
食べたばかりだから、お腹がいっぱい。（剛吃完，所以很飽。）
```

### 3. 概數「大約」（較正式・書面）
```
3人ばかり集まった。（聚集了大約三個人。）
```

## 組合出現時的讀法

```
彼は毎日スマホばかり見ていて、私の話も全然聞いてくれない。
```
- ばかり → 淨是盯著手機（過度）
- も → 「我說的話」追加進「什麼都不做」的範圍
- 全然〜ない → 「一點都不」（全然 強化否定）

```
財布に1000円しかなくて、コンビニでおにぎりだけ買った。
```
- しかない → 只有1000圓（遺憾，不夠多）
- だけ → 只買了飯糰（中性事實，就買這樣而已）
