# 動詞變化型

日文動詞分三類：**五段（う動詞）**、**一段（る動詞）**、**不規則（する・くる）**。先學會分類，才套得了各變化型的規則。延伸：[[adjective-conjugation]]。

## 先辨認動詞屬於哪一類

| 類型 | 怎麼認 | 例 |
|---|---|---|
| 一段（る動詞） | 字尾る，且る前是い段／え段音 | 見る・食べる・起きる・寝る（ね）・教える（おし） |
| 五段（う動詞） | 字尾是う段：く・ぐ・す・つ・ぬ・ぶ・む・る・う | 書く・泳ぐ・話す・待つ・飲む・買う・帰る（る前是え段，仍是五段） |
| 不規則 | 只有兩個，整組背 | する・くる（来る） |

> [!warning] 陷阱
> 字尾る＋前面是い／え段音「看起來像一段」，但這些其實是五段 → 帰る・入る（はいる）・走る（はしる）・要る（いる）・知る（しる）・切る（きる）。同音不同類：切る（きる・五段）vs 着る（きる・一段）。分不出來就查字典。

## 五段的段位原理

五段動詞變化時，字尾在「自己那一行」的あ→お之間移動。以 **書く（か行：か・き・く・け・こ）** 為例：

| 五十音段 | 書く 變成 | 接哪些變化型 |
|---|---|---|
| あ段 | 書**か** | ない形・受身・使役（書かない／書かれる／書かせる） |
| い段 | 書**き** | ます形（書きます） |
| う段 | 書**く** | 辞書形（原形，不變） |
| え段 | 書**け** | ば形・可能形（書けば／書ける） |
| お段 | 書**こ** | 意向形（書こう） |

> [!warning] 換動詞就換它自己那一行
> 待つ→た行（た・ち・つ・て・と）、飲む→ま行（ま・み・む・め・も）。**う 結尾的動詞 a段 用「わ」**：買う → 買わない（不是買あない）。て形／た形另外走音便（見下）。

## 按變化型

每個變化型都分五段／一段／不規則三種變法。

### ない形（否定）「不做～」

| 類型 | 規則 | 例 |
|---|---|---|
| 五段 | あ段 ＋ ない | 書く→書か**ない**／飲む→飲まない／待つ→待たない／買う→買わない（わ） |
| 一段 | 去る ＋ ない | 食べる→食べ**ない**／起きる→起きない |
| 不規則 | 單獨記 | する→し**ない**／くる→こ**ない** |

### ます形（禮貌肯定）「做～（禮貌）」；ません＝禮貌否定

| 類型 | 規則 | 例 |
|---|---|---|
| 五段 | い段 ＋ ます | 書く→書き**ます**／飲む→飲みます／待つ→待ちます |
| 一段 | 去る ＋ ます | 食べる→食べ**ます**／起きる→起きます |
| 不規則 | 單獨記 | する→し**ます**／くる→き**ます** |

### て形（連接／請求／進行）

| 類型 | 規則 | 例 |
|---|---|---|
| 五段（く） | く → いて | 書く→書い**て** |
| 五段（ぐ） | ぐ → いで | 泳ぐ→泳い**で** |
| 五段（す） | す → して | 話す→話し**て** |
| 五段（む・ぬ・ぶ） | → んで | 飲む→飲ん**で**／遊ぶ→遊ん**で** |
| 五段（る・う・つ） | → って | 帰る→帰っ**て**／買う→買っ**て**／待つ→待っ**て** |
| 一段 | 去る ＋ て | 食べる→食べ**て**／起きる→起き**て** |
| 不規則 | 單獨記 | する→し**て**／くる→き**て** |

> [!warning] 例外：行く → 行って（不是行いて）

### た形（過去式）「做了～」；規則同て形，て換成た

| 類型 | 規則 | 例 |
|---|---|---|
| 五段（く） | く → いた | 書く→書い**た** |
| 五段（ぐ） | ぐ → いだ | 泳ぐ→泳い**だ** |
| 五段（す） | す → した | 話す→話し**た** |
| 五段（む・ぬ・ぶ） | → んだ | 飲む→飲ん**だ**／遊ぶ→遊ん**だ** |
| 五段（る・う・つ） | → った | 帰る→帰っ**た**／買う→買っ**た**／待つ→待っ**た** |
| 一段 | 去る ＋ た | 食べる→食べ**た**／起きる→起き**た** |
| 不規則 | 單獨記 | する→し**た**／くる→き**た** |

> [!warning] 例外：行く → 行った（不是行いた）

### ば形（條件）「如果做～的話」

| 類型 | 規則 | 例 |
|---|---|---|
| 五段 | え段 ＋ ば | 書く→書け**ば**／飲む→飲めば／待つ→待てば |
| 一段 | 去る ＋ れば | 食べる→食べれ**ば**／起きる→起きれ**ば** |
| 不規則 | 單獨記 | する→すれ**ば**／くる→くれ**ば** |

### 可能形（能力）「做得到～／能做～」

| 類型 | 規則 | 例 |
|---|---|---|
| 五段 | え段 ＋ る | 書く→書け**る**／飲む→飲める／待つ→待てる |
| 一段 | 去る ＋ られる | 食べる→食べ**られる**／起きる→起き**られる** |
| 不規則 | 單獨記 | する→**できる**／くる→こ**られる** |

> [!warning] する 的可能形是 できる，不是しられる。

### 意向形（意志／邀請）「（我）要做～吧／一起做～吧」

| 類型 | 規則 | 例 |
|---|---|---|
| 五段 | お段 ＋ う | 書く→書こ**う**／飲む→飲もう／待つ→待とう |
| 一段 | 去る ＋ よう | 食べる→食べ**よう**／起きる→起き**よう** |
| 不規則 | 單獨記 | する→し**よう**／くる→こ**よう** |

### 受身形（被動）「被～／受到～」

| 類型 | 規則 | 例 |
|---|---|---|
| 五段 | あ段 ＋ れる | 書く→書か**れる**／飲む→飲まれる／待つ→待たれる |
| 一段 | 去る ＋ られる | 食べる→食べ**られる**／起きる→起き**られる** |
| 不規則 | 單獨記 | する→さ**れる**／くる→こ**られる** |

> [!warning] 一段的可能形和受身形都是られる，靠語境判斷。

### 使役形（使役）「讓～做／叫～做」

| 類型 | 規則 | 例 |
|---|---|---|
| 五段 | あ段 ＋ せる | 書く→書か**せる**／飲む→飲ませる／待つ→待たせる |
| 一段 | 去る ＋ させる | 食べる→食べ**させる**／起きる→起き**させる** |
| 不規則 | 單獨記 | する→さ**せる**／くる→こ**させる** |

## 按詞類（全變化型對照）

### 五段（書く・泳ぐ・話す・飲む・待つ・買う）

| 變化型（用途） | 字尾落點 | 例 |
|---|---|---|
| 辞書形（原形） | う段（不變） | 書く／泳ぐ／話す／飲む／待つ／買う |
| ない形（否定） | あ段 ＋ ない | 書かない／泳がない／話さない／飲まない／待たない／買わない（わ） |
| ます形（禮貌） | い段 ＋ ます | 書きます／泳ぎます／話します／飲みます／待ちます／買います |
| て形（連接） | 音便 | 書いて／泳いで／話して／飲んで／待って／買って |
| た形（過去） | 音便（て→た） | 書いた／泳いだ／話した／飲んだ／待った／買った |
| ば形（條件） | え段 ＋ ば | 書けば／泳げば／話せば／飲めば／待てば／買えば |
| 可能形（能力） | え段 ＋ る | 書ける／泳げる／話せる／飲める／待てる／買える |
| 意向形（意志） | お段 ＋ う | 書こう／泳ごう／話そう／飲もう／待とう／買おう |
| 受身形（被動） | あ段 ＋ れる | 書かれる／泳がれる／話される／飲まれる／待たれる／買われる |
| 使役形（使役） | あ段 ＋ せる | 書かせる／泳がせる／話させる／飲ませる／待たせる／買わせる |

> [!warning] て形／た形音便：く→いて／いた・ぐ→いで／いだ・す→して／した・む/ぬ/ぶ→んで／んだ・る/う/つ→って／った。例外 行く → 行って／行った。

### 一段（食べる・見る・起きる）

| 變化型（用途） | 規則 | 例 |
|---|---|---|
| 辞書形（原形） | 原形（不變） | 食べる／見る／起きる |
| ない形（否定） | 去る ＋ ない | 食べない／見ない／起きない |
| ます形（禮貌） | 去る ＋ ます | 食べます／見ます／起きます |
| て形（連接） | 去る ＋ て | 食べて／見て／起きて |
| た形（過去） | 去る ＋ た | 食べた／見た／起きた |
| ば形（條件） | 去る ＋ れば | 食べれば／見れば／起きれば |
| 可能形（能力） | 去る ＋ られる | 食べられる／見られる／起きられる |
| 意向形（意志） | 去る ＋ よう | 食べよう／見よう／起きよう |
| 受身形（被動） | 去る ＋ られる | 食べられる／見られる／起きられる |
| 使役形（使役） | 去る ＋ させる | 食べさせる／見させる／起きさせる |

> [!warning] 可能形和受身形同形（られる），靠語境判斷。口語可能形常「ら抜き」：食べれる・見れる。

### 不規則（する／くる）

| 變化型（用途） | する | くる（来る） |
|---|---|---|
| 辞書形（原形） | する | くる |
| ない形（否定） | しない | こない |
| ます形（禮貌） | します | きます |
| て形（連接） | して | きて |
| た形（過去） | した | きた |
| ば形（條件） | すれば | くれば |
| 可能形（能力） | できる | こられる |
| 意向形（意志） | しよう | こよう |
| 受身形（被動） | される | こられる |
| 使役形（使役） | させる | こさせる |

> [!warning] する 的可能形是 できる，不是しられる。くる 的讀音整組在變（こ／き／く），不是只換字尾。

## 肯定／否定 × 時制

普通形＝常體（朋友・家人・日記）；丁寧形＝です・ます體（對外・正式）。否定的「過去」是 ない→なかった、ません→ませんでした。

### 五段（書く）

| 時制・文體 | 肯定 | 否定 |
|---|---|---|
| 現在・普通 | 書く | 書か**ない** |
| 現在・丁寧 | 書き**ます** | 書き**ません** |
| 過去・普通 | 書い**た** | 書か**なかった** |
| 過去・丁寧 | 書き**ました** | 書き**ませんでした** |

### 一段（食べる）

| 時制・文體 | 肯定 | 否定 |
|---|---|---|
| 現在・普通 | 食べる | 食べ**ない** |
| 現在・丁寧 | 食べ**ます** | 食べ**ません** |
| 過去・普通 | 食べ**た** | 食べ**なかった** |
| 過去・丁寧 | 食べ**ました** | 食べ**ませんでした** |

### 不規則（する）

| 時制・文體 | 肯定 | 否定 |
|---|---|---|
| 現在・普通 | する | し**ない** |
| 現在・丁寧 | し**ます** | し**ません** |
| 過去・普通 | し**た** | し**なかった** |
| 過去・丁寧 | し**ました** | し**ませんでした** |

### 不規則（くる・来る）

| 時制・文體 | 肯定 | 否定 |
|---|---|---|
| 現在・普通 | くる | **こ**ない |
| 現在・丁寧 | **き**ます | **き**ません |
| 過去・普通 | **き**た | **こ**なかった |
| 過去・丁寧 | **き**ました | **き**ませんでした |

### 其他變化型怎麼變否定（以 書く 為例）

| 變化型 | 肯定 | 否定 |
|---|---|---|
| て形 | 書いて | 書か**なくて**／書か**ないで** |
| ば形 | 書けば | 書か**なければ** |
| 可能形 | 書ける | 書け**ない** |
| 受身形 | 書かれる | 書かれ**ない** |
| 使役形 | 書かせる | 書かせ**ない** |

> [!warning] て形否定有兩種：〜なくて＝原因／狀態（書かなくて困る）；〜ないで＝請求／不做某事（書かないでください）。可能・受身・使役本身是一段型，否定一律「去る ＋ ない」。
