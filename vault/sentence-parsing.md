# 句子解析：主語・受語・修飾

日文不靠位置決定誰是主語，靠**助詞**標角色。讀懂助詞就能拆開任何句子。延伸：[の 接續](no-nominalization.md)、[自他動詞](jidoushi-tadoushi.md)。

## 格助詞 = 角色標籤

| 助詞 | 角色 | 問法 | 例 |
|---|---|---|---|
| **が** | 主語（做動作的） | 「誰が？」 | **猫が**魚を食べた（貓吃了魚） |
| **は** | 話題主語（加對比語感） | 「誰は？」 | **猫は**魚を食べた（至於貓，它吃了魚） |
| **を** | 直接受語（被動的） | 「何を？」 | 猫が**魚を**食べた |
| **に** | 對象・目的地・時間 | 「何に？どこに？いつ？」 | 友達**に**話す／駅**に**着く |
| **で** | 場所（做動作的地方）・手段 | 「どこで？何で？」 | 図書館**で**読む／電車**で**来た |
| **から** | 起點・來源 | 「どこから？」 | 東京**から**来た |
| **まで** | 終點 | 「どこまで？」 | 駅**まで**歩く |

> [!WARNING]
> **位置不決定角色**。猫が魚を食べた 和 魚を猫が食べた 意思相同，只是強調不同。認助詞，不認位置。

## 解析三步驟

1. **找動詞**（在最後面）
2. **找主語**（が / は 前面那個名詞）
3. **找受語**（を 前面那個名詞）

### 示範

```
先生は　学生を　教室に　入れた
```

- 動詞：入れた（いれた）
- 主語：先生**は** → 先生做動作
- 受語：学生**を** → 學生被放入
- 方向：教室**に** → 目的地是教室
- 讀法：「老師把學生放進教室」

```
学生が　教室に　入った
```

- 動詞：入った（はいった）
- 主語：学生**が** → 學生自己進去
- 方向：教室**に**
- 讀法：「學生（自己）進了教室」

## 修飾語：永遠在被修飾語的左邊

英文把修飾語放在名詞右邊（the book **that I bought yesterday**）；日文全部放在**左邊**。

| 日文 | 修飾結構 | 中文 |
|---|---|---|
| **昨日買った**本 | [昨日買った] → 本 | 昨天買的書 |
| **駅の近くにある**カフェ | [駅の近くにある] → カフェ | 在車站附近的咖啡廳 |
| **先生が褒めた**学生 | [先生が褒めた] → 学生 | 老師稱讚的學生 |
| **彼女が作った**料理 | [彼女が作った] → 料理 | 她做的料理 |

規則只有一條：**動詞或形容詞出現在名詞前面 → 它在修飾那個名詞，不是句子的主動詞。**

> [!WARNING]
> 陷阱：先生が学生を褒めた（老師稱讚了學生）← が・をここ是句子結構
> vs [先生が褒めた]学生（被老師稱讚的學生）← 動詞前有名詞，整組是修飾語
> 區別在於動詞是**句末主動詞**還是**名詞前的修飾語**。

## 修飾語層層疊加

一個名詞可以被整個子句修飾。讀的策略：**先找最後的名詞，再往左讀誰在描述它。**

```
[先生が学生に教えた]内容
```
→ 「先生が学生に教えた」整句修飾「内容」→「老師教給學生的內容」

```
[昨日図書館で借りた]本を読んでいる
```
→ [昨日図書館で借りた] 修飾 本 → を → 受語
→ 讀法：「（我）正在讀昨天在圖書館借的書」

## 主語省略（日文常態）

日文主語常被省略，要靠語境補回來：

- 疲れた。（（我）累了。）
- どこ行くの？（（你）去哪？）
- 宿題した？（（你）做（了）作業嗎？）

讀到省略主語時，問「誰做了這件事？語境裡說話的是誰？」

## 快速拆句流程

```
1. 找句末動詞
2. 往左掃助詞：が/は → 主語；を → 受語；に/で/から/まで → 其他修飾
3. 遇到「動詞/形容詞 ＋ 名詞」組合 → 前面是修飾語，括起來
4. 補回被省略的主語（靠語境）
```

### 綜合練習句

```
彼女が昨日公園で拾った財布を警察に届けた。
```

拆解：
- 句末動詞：届けた（交出）
- 主語：彼女**が**
- 受語：財布**を**
- 財布の修飾語：[彼女が昨日公園で拾った]（她昨天在公園撿到的）
- 目的地：警察**に**
- 完整讀法：「她把昨天在公園撿到的錢包交給了警察。」
